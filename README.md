RoboSprouts – AI & Robotics Summer Workshop

RoboSprouts is a full-stack workshop landing page created as part of a technical assignment for Kidrove. The project focuses on building a clean, responsive page for an online AI and Robotics workshop designed for children.

Workshop Overview

Workshop Name: AI & Robotics Summer Workshop
Age Group: 8–14 Years
Duration: 4 Weeks
Mode: Online
Fee: ₹2,999
Start Date: 15 July 2026

Project Features

Responsive landing page inspired by Kidrove’s workshop style
Hero section with workshop title and enroll button
Workshop details and learning outcomes
FAQ section for common questions
Registration form with basic validation
Backend API to handle form submissions

Technology Stack

Frontend

React.js
TypeScript
Tailwind CSS
Vite

Backend

Node.js
Express.js
MongoDB (optional integration)

API Endpoint

POST /api/enquiry

The endpoint accepts registration form data, validates required fields, and returns a success response after submission.

Development Approach

I focused on building reusable React components to keep the code clean and organized. Tailwind CSS was used to quickly implement a responsive layout. The backend was developed using Express.js to keep the API simple and easy to understand.

Future Improvements

With more time, I would improve form validation and error handling, add email notifications for registrations, deploy the project online, and enhance overall performance and accessibility.

Environment variables are excluded from the repository for security reasons.
